How to Use

1. WordPress Shortcode

Use the shortcode anywhere in your WordPress content:

[tiktok_lightbox url="https://www.tiktok.com/@matty.b129/video/7503809700601908488" button_text="Watch Video"]

2. Elementor Widget

Search for "TikTok Lightbox" in the Elementor widget panel and drag it to your page. Configure the settings in the widget panel:

  • TikTok URL: Enter the full TikTok video URL
  • Custom Thumbnail: Upload your own thumbnail image (optional)
  • Button Text: Customize the call-to-action text
  • Styling Options: Control colors, sizes, borders, and animations

3. Direct JavaScript

For developers, you can trigger the lightbox programmatically:

// Open lightbox with video ID TikTokLightbox.open('7503809700601908488'); // Close lightbox TikTokLightbox.close();

4. jQuery Plugin

Use as a jQuery plugin:

$('.my-tiktok-element').tikTokLightbox({ videoId: '7503809700601908488' });
